# healthour-pharmacy
My React App website

* 資策會（資展國際） 畢業專題製作  
* 主題：健康藥局(Healthour)  
* 負責：網站主視覺、Logo繪製  
* 負責頁面：首頁、門市資訊、保健百科、處方籤門市領藥 <br>
網站連結入口： https://portiao.github.io/healthour-pharmacy/

## 安裝工具  
請先安裝 Visual Studio Code，包括以下擴充功能（extensions）：  
* Babel JavaScript
* ES7 React  
* Reactjs code snippets
* clone後，執行npm install 安裝完整套件

##  使用技術
* 前端技術 ： React 、 RWD響應式網頁
* API串接  :
> 行政院環保署 (空氣品質資訊)  
> Google Map  

* 後端技術 ： Node.js 、 MySQL
